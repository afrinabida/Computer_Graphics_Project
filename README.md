Installation Steps
1. Extract the File
Unzip the downloaded archive.
Copy or extract all files to the D:\ drive.
2. Open the Project
Go to the folder: D:\Hills
Launch Code::Blocks.
Open the project file named Hills.cbp.
3. Compile & Execute
Click Build to compile the project successfully.
Press Run to launch the application.
Scene 1
Start by clicking the Left Mouse Button (LMB).
Once the car leaves the screen, press j to move forward to the next scene.
Scene 2
Begin the scene with LMB click.
As night arrives and the bonfire appears:
Look at the boy’s pocket to notice a phone notification symbol.
Message Interaction Keys:
i → Open Message 1
o → Open Message 2
p → Open Final Message
After reading all messages, press j to proceed to Scene 3.
Scene 3
This section runs automatically without any user input.
Fast Scene Controls
l → Directly go to Scene 2 from Scene 1
q → Jump to Scene 3 (Part 1)
a → Jump to Scene 3 (Part 2)
z → Jump to Scene 3 (Part 3)
Extra Controls (Scene 2 Features)
RMB → Restart the scene
k → Stop vehicle (Jeep)
c → Turn on night mode
v → Switch to day mode
s → Activate sun
d → Deactivate sun
m → Activate moon
n → Deactivate moon
w → Start snowfall
e → Stop snowfall
r → Trigger phone ringing
t → Stop phone sound
f → Start bonfire
g → Stop bonfire
Note: All keyboard inputs must be entered in lowercase letters only 🔤
